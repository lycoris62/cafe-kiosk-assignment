플로우 차트 도형 
* 출력 : 파랑     
* 입력 : 초록     
* 분기 : 노랑     
* 로직 : 보랑     

플로우 차트 배경 
* 빨강 : 데코레이터 패턴 & 정적 팩토리 패턴 
* 연두 : 커맨드 패턴
* 노랑 : 책임의 사슬 (어댑터) 패턴
* 그외에 싱글톤 패턴도 사용했습니다.

자세한 플로우 차트 및 설계에 대한 설명은 [여기](https://github.com/lycoris62/cafe-kiosk-assignment/wiki/%EC%84%A4%EA%B3%84%EB%8F%84)에 상세히 적혀 있습니다.
